Lista de Definiciones

HTML

Lenguaje de marcado utilizado para crear páginas web.

CSS

Lenguaje de hojas de estilo usado para definir la apariencia de una página web.
Lista de Elementos Importantes

  • Encabezados: Usados para definir títulos y subtítulos.

  • Listas: Permite organizar contenido en viñetas o numeraciones.

  • Enlaces:Conectan diferentes páginas o recursos.
